# Smart OCR & Text Translator

This is a Python-based web application that extracts text from images and translates it into multiple languages.

## Technologies Used
- Python
- Flask
- EasyOCR
- Googletrans
- HTML, CSS, JavaScript

## Features
- Upload image and extract text using OCR
- Translate extracted text into multiple languages
- Translate manually entered text
- Simple web-based interface

## How to Run
1. Install required libraries
2. Run `python app.py`
3. Open browser and go to `http://127.0.0.1:5000`
